## 预处理

1. 递归遍历解压
   - 压缩包里面又包含压缩包, 所以需要重复这个过程直到全部解压完毕
   - 解压同名问题, 如有`t1.zip`和 `t1.rar`, 应该解压成 `ti_zip` 和 `t1_rar`文件夹,避免文件夹合并
1. 清理解压失败的文件, 空文件夹, 垃圾数据等
   - 解压时会碰见压缩包数据损坏, 需要密码等情况,
   - 原文件夹中存在垃圾数据, 未下载完的数据等等, 不能正常使用的应该清理掉
1. 清理相同的文件
   - sha1 后直接清理掉
